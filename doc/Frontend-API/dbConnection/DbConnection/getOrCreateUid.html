getOrCreateUid method

Future<String> getOrCreateUid()

Ruft die vorhandene Benutzer-ID (UID) aus dem lokalen Speicher ab oder erstellt eine neue, falls sie nicht existiert.

Dadurch wird sichergestellt, dass der Benutzer eine dauerhafte Identität über Sitzungen hinweg hat. Gibt die eindeutige ID des Benutzers zurück.

Implementation

Future<String> getOrCreateUid() async {
  final prefs = await SharedPreferences.getInstance();
  var uid = prefs.getString('UID');
  if (uid == null) {
    uid = _uuid.v4();
    await prefs.setString('UID', uid);
  }
  return uid;
}
