DbConnection

DbConnection class

Stellt eine Verbindung zur Supabase-Datenbank her, um alle spielbezogenen Daten zu verwalten.

Diese Klasse kapselt alle Methoden, die für die Interaktion mit dem Backend benötigt werden, einschließlich des Abrufens von Kartendaten, der Benutzerverwaltung, dem Erstellen und Beitreten von Spielen und der Verwaltung des Spielzustands.
